# Simple Calculator

A beginner-friendly Command Line Calculator built using Python.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Multiple calculations using loop
* User input handling

## Technologies Used

* Python

## How to Run

1. Install Python
2. Open terminal or command prompt
3. Run the file using:

```bash
python calculator.py
```
